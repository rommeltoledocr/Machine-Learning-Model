# Student Dropout Risk Prediction Model

Este proyecto busca desarrollar un modelo de **Machine Learning supervisado enfocado en regresión** para predecir el **riesgo de deserción estudiantil (dropout)** a partir de ciertas **características iniciales del estudiante**.

El objetivo es identificar patrones tempranos que permitan estimar qué tan propenso es un estudiante a abandonar sus estudios, utilizando información académica, demográfica y socioeconómica disponible desde etapas iniciales.

A través del análisis de datos y el entrenamiento de modelos predictivos, se busca generar una herramienta que ayude a comprender qué factores tienen mayor influencia en el riesgo de abandono escolar y facilite una detección temprana para futuras intervenciones.

## Dataset

El modelo será entrenado utilizando el dataset **[Predict Students' Dropout and Academic Success](https://archive.ics.uci.edu/dataset/697/predict+students+dropout+and+academic+success)**, el cual contiene información de **4,424 estudiantes y 36 variables**, incluyendo factores académicos, demográficos y socioeconómicos relacionados con el desempeño y abandono escolar.
